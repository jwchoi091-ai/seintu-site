# 수사해당가든 운영 원장 (사용 안내)

`수사해당가든_운영원장.xlsx` = 재고·매입/매출·협력농가·교육을 한 파일에서 관리하는 **단일 원장**입니다.

## 여는 법
- **구글시트로 쓰기**: 이 엑셀 파일을 구글 드라이브에 끌어다 놓으면 자동으로 구글시트로 열립니다.
- **엑셀로 쓰기**: 그냥 더블클릭.

> 구글 계정 연결은 지금 필요 없습니다. 나중에 홈페이지와 실시간 연동(파트 5)할 때 연결합니다.

## 시트 구성
| 시트 | 용도 |
|---|---|
| **안내** | 사용법·원칙 요약 |
| **재고** | 나무 등록(핵심). 대형목=1주 1행(개체), 묘목=규격묶음+수량(로트) |
| **거래** | 매입·매출 기록 → 대시보드로 자동 집계 |
| **협력농가** | 위탁·매입해 파는 협력 농가 목록 |
| **교육** | 교육 회차·인원·시간 (홈페이지 '누적 교육 인원' 연동 예정) |
| **대시보드** | 재고 현황·기간별 매출/매입/마진·교육 누계 자동 계산 |
| **견적서** | 품목만 채우면 합계 자동 계산되는 견적서 양식 |

## 핵심 원칙 (나무 재고)
- 년생으로 규격을 추정하지 말고 **R(목대)·수고·지하고를 실측값**으로 적기.
- 나무는 자라므로 봄·가을 등 정기적으로 규격과 **최근실측일** 갱신.
- **소유구분(자가/협력농가)** 을 채우면 협력농가 나무 판매 시 마진 계산의 기초가 됨.

## 대시보드 사용
- 노란 칸 **기준연도**만 바꾸면 분기/반기/연 집계가 바뀝니다.
- 그 외 숫자는 재고·거래·교육 시트에 데이터를 넣으면 자동 갱신.

## 세금계산서 관련 (확인 필요)
- 묘목·화훼 등 미가공 농산물은 부가세 **면세**일 수 있어, 세금계산서가 아니라 **계산서**를 발행합니다.
- 원예자재·식재 시공은 과세일 수 있어 겸업이 됩니다 → **세무사 확인 권장**.
- 계산서 발행은 국세청 **홈택스**(무료·공인) 이용. 견적서는 이 파일의 '견적서' 시트로.

## 앞으로 (파트 5에서 홈페이지 연동)
- 재고 시트의 판매상태 → 홈페이지 상품 '판매중/예약/품절' 배지 자동 반영
- 교육 시트 → 홈페이지 농장 일정 + '누적 교육 인원' 자동 갱신
- Netlify 견적 폼 제출 → 거래/문의로 자동 수집
- 견적서 자동 생성(PDF)
